# JavaScript DSA (Data Structures & Algorithms)

A comprehensive collection of Data Structures and Algorithms problems solved in JavaScript, ranging from beginner to advanced level. This repository is designed to help you improve your JavaScript core knowledge and problem-solving skills.

## üìö Problem Categories

### üü¢ Beginner Level
Perfect for those starting their DSA journey. These problems cover fundamental concepts and basic algorithms.

1. **[Reverse a String](./beginner/01_reverse_string.js)**
   - Learn string manipulation techniques
   - Multiple solutions: built-in methods, loops, recursion
   - Time Complexity: O(n)

2. **[Find Maximum Number in Array](./beginner/02_find_max_number.js)**
   - Array traversal and comparison
   - Solutions using Math.max, loops, and reduce
   - Time Complexity: O(n)

3. **[Palindrome Check](./beginner/03_palindrome_check.js)**
   - String comparison and two-pointer technique
   - Handles special characters and case sensitivity
   - Time Complexity: O(n)

### üü° Intermediate Level
Build upon your foundation with more complex algorithms and problem-solving techniques.

1. **[Binary Search](./intermediate/01_binary_search.js)**
   - Efficient searching in sorted arrays
   - Both iterative and recursive implementations
   - Time Complexity: O(log n)

2. **[Two Sum Problem](./intermediate/02_two_sum.js)**
   - Hash map optimization technique
   - Brute force vs optimal solution comparison
   - Time Complexity: O(n) with hash map

3. **[Merge Sort](./intermediate/03_merge_sort.js)**
   - Divide and conquer algorithm
   - Learn recursive sorting techniques
   - Time Complexity: O(n log n)

### üî¥ Advanced Level
Challenge yourself with complex algorithms involving dynamic programming, graphs, and optimization problems.

1. **[Longest Common Subsequence](./advanced/01_longest_common_subsequence.js)**
   - Dynamic programming classic problem
   - Multiple approaches: recursion with memoization, bottom-up DP, space-optimized
   - Time Complexity: O(m * n)

2. **[Graph Depth First Search (DFS)](./advanced/02_graph_dfs.js)**
   - Graph traversal algorithms
   - Both recursive and iterative implementations
   - Includes BFS for comparison
   - Time Complexity: O(V + E)

3. **[0/1 Knapsack Problem](./advanced/03_knapsack_problem.js)**
   - Classic optimization problem
   - Dynamic programming with backtracking
   - Multiple space-time trade-off solutions
   - Time Complexity: O(n * W)

## üìñ Learning Path

**For Beginners:**
1. Start with beginner-level problems
2. Understand the problem statement thoroughly
3. Try solving it yourself before looking at solutions
4. Study all solution approaches to learn different techniques
5. Run the code and verify test cases

**For Intermediate Learners:**
1. Focus on understanding time and space complexity
2. Compare different solution approaches
3. Try to optimize brute force solutions
4. Practice implementing both iterative and recursive solutions

**For Advanced Learners:**
1. Master dynamic programming concepts
2. Learn graph algorithms and traversals
3. Understand optimization techniques
4. Practice solving problems in multiple ways

## üéØ Problem-Solving Tips

- **Understand the problem:** Read carefully and identify inputs, outputs, and constraints
- **Plan your approach:** Think of multiple solutions before coding
- **Start simple:** Begin with a brute force solution, then optimize
- **Test thoroughly:** Run your code with various test cases including edge cases
- **Analyze complexity:** Always consider time and space complexity
- **Learn from others:** Study different approaches to the same problem
